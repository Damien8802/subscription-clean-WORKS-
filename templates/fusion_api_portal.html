📚 Документация API

Аутентификация

curl -X GET https://fusionapi.ru/v1/crm/customers \
  -H "X-API-Key: YOUR_API_KEY"

API Эндпоинты

GET /api/v1/crm/customers - список клиентов
POST /api/v1/crm/deals - создать сделку
GET /api/v1/fincore/entries - бухгалтерские проводки
GET /api/v1/teamsphere/tasks - задачи TeamSphere

🤖 AI Agents Эндпоинты

GET /api/fusion/agents - список агентов
POST /api/fusion/agents - создать агента
PUT /api/fusion/agents/:id - обновить агента
DELETE /api/fusion/agents/:id - удалить агента
POST /api/fusion/agents/:id/chat - чат с агентом
