📚 Документация API для партнеров

API Endpoints

GET /api/partner/stats

Получить статистику партнера

curl -X GET http://localhost:8080/api/partner/stats

GET /api/partner/earnings

Получить доходы партнера

curl -X GET http://localhost:8080/api/partner/earnings

Пример ответа

{
  "status": "success",
  "data": {
    "revenue": 0,
    "referrals": 0,
    "commission": 0,
    "active": true,
    "level": "basic"
  }
}
