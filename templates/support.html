❓ Часто задаваемые вопросы

🔹 Как создать API ключ?
Перейдите в профиль, раздел "API ключи", нажмите "Создать ключ". Ключ покажется только один раз — обязательно сохраните его.
🔹 Какие способы оплаты?
Мы принимаем банковские карты, USDT, Bitcoin, СБП и CryptoBot.
🔹 Как сменить тариф?
Перейдите в раздел "Мои подписки" и выберите новый тариф.
🔹 Как отозвать API ключ?
В профиле в разделе API ключей нажмите кнопку "Отозвать" рядом с нужным ключом.
🔹 Не приходит письмо на почту?
Проверьте папку "Спам". Если письма нет — напишите нам в чат поддержки.
